# Devuelve un DataFrame con 8 métricas:
# 1. count
# 2. mean
# 3. std
# 4. min
# 5. 25% (primer cuartil)
# 6. 50% (mediana/segundo cuartil)
# 7. 75% (tercer cuartil)
# 8. max

    # Sirve para detectar si faltan datos.
    # Resultado: entero, ej. 1000

    # mean: promedio de los valores.
    # Sirve para referencia central de los datos.
    # Resultado: float, ej. 7.34
